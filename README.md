# pythonGame
attempt to create a game

# Простая игра "Угадай число"

Файлы:
- main.py — точка входа.
- game.py — логика игры.

Как запустить в Visual Studio Code:
1. Открой папку `d:\Programming\pythonGame` в VS Code.
2. Убедись, что установлен Python и выбран интерпретатор (Ctrl+Shift+P → Python: Select Interpreter).
3. Запусти отладку (F5) или выполни `python main.py` в терминале.
